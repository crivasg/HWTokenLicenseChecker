Detailed overview on the HWU usage of the license server.

Used instead of the tools provided at the HyperWorks installation to check the usage of the licenses.

Requirements

System.Data.Sqlite (http://system.data.sqlite.org)
.Net 3.5 (or later)
Altair Hyperworks 10.1, 11.0
The following enviroment variables are needed:
	LMX_LICENSE_PATH: Assigned to the server with the HW licenses. i.e.6200@10.0.1.10
	ALTAIR_HOME: Path to the Altair home. i.e W:\Program Files\Altair\11.0
